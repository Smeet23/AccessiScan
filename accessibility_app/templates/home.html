About This Tool

Our Accessibility Analyzer uses Claude AI to scan web pages for accessibility issues. It checks for common WCAG (Web Content Accessibility Guidelines) violations and provides suggestions for improvement. This helps make the web more accessible for people with disabilities.

What We Check For:

  • Color contrast issues
  • Missing alt text on images
  • Improper heading structure
  • Form accessibility problems
  • Keyboard navigation barriers
  • And many more accessibility concerns
